# Question Format Guide

## Rules

1. **NEVER ask questions directly in chat** — all questions go in dedicated `.md` files
2. Use multiple choice format with `[Answer]:` tags
3. **Always include "Other" as the last option**
4. Each option separated by a blank line

## Format

```markdown
# [Phase] Questions

## Question 1
[Clear, specific question]

A) [Option]

B) [Option]

C) [Option]

X) Other (please describe after [Answer]: tag below)

[Answer]: 
```

## User Responds By

Filling in the letter after `[Answer]:` — e.g., `[Answer]: B`

## After Receiving Answers

1. Read the question file
2. Validate all questions are answered
3. Check for contradictions between answers
4. If contradictions found, create a clarification file and wait for resolution
5. Proceed only when all answers are clear and consistent
